My resume

Barren Nebula

My first published game, available for Android OS on Google Play, and Windows 10 Universal on the Microsoft Store. It's an on-rails shooter with a 3rd person perspective, where the player must defeat the enemies in order to collect items and proceed. The game was conceived to be a series of small episodes, each containing one level featuring several difficulty levels and unlockables, in order to keep the players engaged. The first episode was released in November 2016, and a few more are coming on their way. The concept/challenge behind this game was creating a small, fun game using mostly Creative Commons Assets.

Slot Machine in Phaser

A simple slot machine I developed while learning Phaser. It is a basic 3-reel 5-lined slot machine, with randomly generated slots, like we see in modern digital slot machines. In this example the user can add an unlimited amount of credits, by using the big plus sign button. The user can choose to play from 1 to 5 lines using the arrow buttons, and then use the big round button to spin the reels. This is just a small "assignment" I came up with in order to learn Phaser. There is a lot of room for improvements, as the assets were developed for a more complex machine, but I decided to keep it simple for a first non-tutorial based project. The amazing graphics assets were found in Unity's Asset Store.

Guardians of Tech

A tower defense game, developed in the course of Digital Games, at FEUP. The player must carefuly lay his defenses, using circuits to connect the different towers to the main power unit, and defend the last remnants of a human technology that will allow the rebuilding of Earth after an Alien apocalypse.

Hit The Beat

A rythm-based videogame developed for Microsoft, in the course of LGP. It was developed using the Unity engine as a Universal App, running in all Windows devices. Gameplay experience on a smartphone or a tablet is different than in the computer, which also features Xbox One/360 controller support.

IdeaTouch

IdeaTouch, a very simple and easy to use ideation/idea management platform, which is result driven. IdeaTouch emulates a multi-touch system using individual devices. Each person may use their smartphone, tablet or computer to access IdeaTouch and contribute with their ideas and opinions on a problem under discussion.
